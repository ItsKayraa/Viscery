# Viscery
Viscery is a open-source framework to develop SurfaceGUIs in SCP: Roleplay. With Viscery, you can create GUIs with just a visc, rather than putting Instance.news one by one.

## Instructions

1. Copy the Addon Code [here](viscery.lua)
2. Open an editor, this can be Notepad for your device or anywhere you can edit text.
3. Do your configurations of Viscery.
4. Copy the configurated code.
5. Paste it into server addons.
6. Add Addon.
7. To use it, look to API from [here](https://viscery.github.io).
